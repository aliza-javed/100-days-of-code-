# 🌟 Daily Commit Tracker

A Python application to track daily coding progress for the **100 Days of Code Challenge**.

## ✨ Features

- ✅ Log daily coding progress
- ✅ Track project names and goals
- ✅ Auto-increment day counter
- ✅ Maintain streak count
- ✅ Save progress to JSON file
- ✅ View all past entries
- ✅ Delete entries if needed
- ✅ Motivational messages
- ✅ Milestone celebrations

## 🛠️ Technology Stack

- **Language:** Python 3.x
- **Data Storage:** JSON (local file)
- **Libraries:** datetime, json, os, random
