## 📖 Overview

This project presents an **Exploratory Data Analysis (EDA)** of the **US Top 50 Universities 2026** dataset.

The goal is to explore relationships between university rankings, research performance, employment outcomes, international student ratios, university age, and institution type using Python and data visualization techniques.

This project is part of my **100 Days of Code** challenge.

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/aliza-javed/100-days-of-code-.git
```

Open the project

```bash
cd day_US_Top_50_Universities_2026
```

Create a virtual environment

### Windows

```bash
python -m venv venv
```

Activate it

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
